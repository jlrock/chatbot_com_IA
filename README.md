# **Mathbot, o chatbot professor de matemática**
## Instruções para executar o programa:
1. Ao clonar este repositório para a sua máquina local, você deve renomear o arquivo ".env.example" para ".env"
2. Após isso, você deve acessar o site [Google AI Studio](https://aistudio.google.com/) e criar uma chave de API para esse projeto
3. Com isso, você cola a sua chave dentro da variável do arquivo ".env"
4. Finalmente, basta executar o arquivo principal "MathBot.py" na sua IDE e usar o chatbot no seu terminal
